Научная методика расчёта
По статье Литвин Р.А. "Методика расчёта параметров амортизатора" (2018)

📐 Геометрические размеры:

  • Разжатый = штатный + лифт
  • Сжатый = штатный + 50% лифта
  • Нейтральное = (сжатый + разжатый) / 2

🔬 Формулы демпфирования:

  • • K = 2ψ√(M·C) — коэф. сопротивления
  • • ψ = δ/π = 0.15...0.30 — апериодичность
  • • K_отб = √(2K/(1+B)) — коэф. отбоя
  • • K_сж = B·K_отб — коэф. сжатия
  • • F = K·V — сила сопротивления
  • • f = (1/2π)√(C/M) — собств. частота

📊 Параметры:

  • ψ — коэффициент апериодичности (затухания)
  • M — масса на колесо, кг
  • C — жёсткость подвески, Н/м
  • B — соотношение усилий сжатие/отбой (4-8)
  • V — скорость поршня, м/с
Жёсткость подвески C:
• Перед: штатные 270 кН/м, мягкие 230 кН/м, жёсткие 352 кН/м
• Зад: 2 листа 187 кН/м, 3 листа 220 кН/м, 5+ листов 297 кН/м
Режимы эксплуатации:
• Город: ψ×0.9, V=0.3 м/с
• Смешанный: ψ×1.0, V=0.42 м/с
• Бездорожье: ψ×1.15, V=0.51 м/с
• Экстрим: ψ×1.3, V=0.6 м/с
